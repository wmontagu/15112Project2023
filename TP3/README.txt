This game is called Algorithmic Ascension. It is a card game in which you fight enemies by using cards. Your goal is to ascend to the top and beat the final boss as fast as possible without dying.
The user should run the TermProjectAlgorithmicAscension.py file in order for the game to work. The other files are either classes or images and will be used inside this python file.
No libraries were installed, (aside from CMU_graphics)
Use the left arrow key to move 1 card from the draw deck into my hand. Use the right arrow key to move 1 card from my draw to discard deck.
Press 'y' to restore energy without ending my turn.
